# Quick ShulkerBox | 快速潜影盒

> 这是一个 Endstone 服务端插件，可以让玩家手持潜影盒对空气交互就可以打开潜影盒，并从中取出、放入物品。

---

## 安装方法

从 Release 中下载对应平台的构建压缩包，解压后将其中的 `dll` / `so` 文件放入你服务端的 `plugins` 文件夹即可。

---
