# Quick ShulkerBox

> An Endstone server plugin that lets players open a shulker box by right-clicking air while holding it, allowing items to be taken out or put in directly.

---

## Installation

Download the build archive for your platform from Releases, extract it, and place the `dll` / `so` file into your server's `plugins` folder.

---

## Known Issues

- Potion-related items lose their effect data when stored or retrieved, including tipped arrows
- Bundles placed into the shulker box from the player's inventory do not preserve their contained items' data
- The shulker box UI does not offer the same flexibility as the vanilla container interface
